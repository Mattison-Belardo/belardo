Choropleth Map : Percentage of Kids in Virginia Counties`

The choropleth map displays the percentage of children enrolled in school in each countie of Virginia. The colors of the polygons are based on the range of values in the domain attribute, with the range value in the first slot of the domain attribute filling the color of the polygon in the first slot of the range attribute. The legend is created using SVG, which is the specification for the map. It also adds titles to the map indicating the percent of kids in each region. The values of each polygon are displayed when the user hovers over the map, and the color and width of the polygon borders (outlines) are determined using the attr() function.

Point Symbol Map : Total Number of Famlies Per US State

This graduated symbol map displays the number of families in each US state. The map is colored in shades of purple, with darker colors representing a higher number of families. The points on the map are circles that vary in size, with larger circles representing higher numbers of families. The legend on the map displays the range of values that are represented by the size and color of the circles. The map also displays the total number of families when you hover over each point. This map provides a quick and easy way to visualize the number of families living in each US state.

Isarithmic Map : Yearly Total Precipitation

The isarithmic map created displays the yearly average total precipitation throughout the state of Iowa. The data used to create the map was obtained from The Iowa Environmental Mesonet (IEM). The map was created using the d3 library, which allows for the creation of interactive maps. The map is composed of polygons which each represent a range of yearly precipitation. The color of each polygon corresponds to its range, with darker colors representing higher precipitation totals. In addition, a legend is provided to better understand the range of values associated with each color. The map provides a visual representation of the yearly precipitation totals in Iowa and can be used to help understand and analyze the effect of precipitation on the environment.

Bivariate Choropleth Map : Total Land Per Area and Total Water Per Area in Virginia County

This bivariate map of Virginia displays the total land and water area per county. The map uses a color scheme that ranges from light gray to dark green, with the lightest color representing the lowest values and the darkest color representing the highest values. The legend explains the color scheme and the hover feature displays the exact values of the two data sets being mapped. The map was created by joining the geometries from the topojson file with the attributes from the csv file, utilizing the GEOID as the key. This allows for an accurate representation of the land and water area in each county of Virginia.
